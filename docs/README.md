# mintalib API Documentation

API reference for [mintalib](https://github.com/furechan/mintalib) — a minimal technical analysis library for Python.

## Modules

- [mintalib](mintalib.md) — package overview
- [mintalib.core](mintalib.core.md) — core calculation routines (`calc_sma`, `calc_ema`, `calc_macd`, ...)
- [mintalib.functions](mintalib.functions.md) — plain functions (`sma`, `ema`, `macd`, ...)
- [mintalib.indicators](mintalib.indicators.md) — composable indicator objects (`SMA`, `EMA`, `MACD`, ...)
- [mintalib.expressions](mintalib.expressions.md) — polars expression factories (`SMA`, `EMA`, `MACD`, ...)
